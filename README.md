# 株式会社興栄コンサルタント kintone導入・活用支援サイト

kintone導入・活用支援を専門とする株式会社興栄コンサルタントの公式Webサイト（デモバージョン）です。

## 🎯 プロジェクト概要

**目的**: 民間企業・自治体向けのkintone導入・伴走支援サービスをわかりやすく訴求し、問い合わせを促進  
**ターゲット**: 業務効率化に課題を持つ民間企業、DX推進を目指す自治体  
**コンセプト**: 60年以上の実績をもつ技術コンサルタントが提供する、現場に寄り添うDX支援

## 💼 主要機能

### トップページ
- 民間企業向け（オレンジ）/ 自治体向け（ブルー）の色分けCTA
- 資料ダウンロード（イエロー）を改行して配置
- 強みセクション（5項目）
- 課題解決セクション（5項目）
- 導入事例3件
- コラム紹介3件

### 民間企業様向けページ
- ファーストビュー：Before/After視覚化
- 共感ゾーン：7つの課題カード
- サービス内容と価格：
  - STEP 0: 無料相談（¥0）
  - STEP 1: 導入支援パック（¥198,000〜）
  - STEP 2: 運用支援プラン（¥30,000/月〜）
  - STEP 3: 内製化支援（¥70,000〜）
- AI関連の課題2項目追加
- 導入事例3件

### 自治体様向けページ
- キャッチコピー：「アナログな「紙」仕事から、住民のための「価値」仕事へ。」
- 活用シーン6カテゴリ：
  1. 住民対応・窓口業務
  2. 申請・手続き業務
  3. 庁内業務・バックオフィスの効率化
  4. 財務・予算管理
  5. 人事・職員管理
  6. 防災・危機管理
- サービス内容：
  1. 自治体まるごとDXボックス（2026年4月〜2027年4月末限定、無償）
  2. カイゼンThanksパック（¥390,000、税別）
  3. サイボウズ災害支援（緊急時無償、平時訓練無償）
- 導入事例3件（A市、B市、C町）

### 導入事例ページ
- タブ切り替え機能（すべて/民間企業/自治体）
- 各事例カード：業種、従業員規模、導入前の課題、導入後の成果

### コラムページ
- カテゴリフィルター（すべて/活用事例/機能紹介/業務改善ヒント/DXトレンド）
- 各記事カード：カテゴリバッジ、タイトル、概要、公開日、タグ

### 会社概要ページ
- 会社情報テーブル
- 5つの強み（実績、技術力、サポート、地域密着、パートナー）
- アクセス情報
- 代表者メッセージは削除済み

## 🚀 ローカル確認方法

### 方法1: Live Server（VS Code拡張機能）
1. VS Codeで「Live Server」拡張機能をインストール
2. `index.html` を右クリック → 「Open with Live Server」

### 方法2: Python簡易サーバー
```bash
# Python 3の場合
python -m http.server 8000

# ブラウザで http://localhost:8000 を開く
```

### 方法3: Node.js http-server
```bash
npx http-server -p 8000
```

## 🌐 デプロイ方法（推奨）

### Netlify（推奨）
1. GitHub/GitLabにリポジトリをプッシュ

2. Vercelで「Import Project」
3. リポジトリを選択 → Deploy

1. GitHubリポジトリの設定 → Pages

3. 数分でデプロイ完了
